## ⚡ Quick Commands Reference

**PostgreSQL:**
```bash
psql -U postgres
CREATE DATABASE lms_db;
```

**Run Migrations:**
```bash
psql -U postgres -d lms_db -f 001_create_users_table.sql
# ... (3 more migrations)
```

**Install & Start:**
```bash
cd backend && npm install && npm run dev
cd frontend && npm install && npm run dev
```

**Access App:**

http://localhost:5173
